🧮 Calculator App (React)

A modern and responsive Calculator Application built using React.js. This app performs basic arithmetic operations with a clean UI and real-time updates using React state management.

🚀 Features
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
🧠 Real-time calculation
🧹 Clear (AC) functionality
⚛️ Built with reusable React components
📱 Fully responsive design

🛠️ Tech Stack
React.js – Component-based UI
HTML5 – Structure
CSS3 – Styling
JavaScript (ES6+) – Logic & operations

🎯 How It Works
Uses React useState to manage input and result
Button clicks update the current expression
Expression is evaluated dynamically
Display component shows real-time output
